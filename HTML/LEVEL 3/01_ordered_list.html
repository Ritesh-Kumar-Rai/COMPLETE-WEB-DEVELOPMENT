The ordered list is used to display the list of items but in a ordered manner

  1. Apple
  2. Orange
  3. Banana

There are types of ordered list:
example:
ol type="1" for 1,2,3,4... it is by default also
ol type="A" for A,B,C,D...
ol type="a" for a,b,c,d...
ol type="I" for UpperCase Roman like I,II,III,IV...
ol type="i" for lowercase Roman like i,ii,iii,iv...
we use type attribute in ol tag
