Responsive Web Design | Media Queries: width, max-width, min-width and combinations of both

Syntax:

    @media  and (condition){
        .container{
            // write styles here
        }
    }

Example:

        @media screen and (width: 720px){
            .container{
                background-color: purple;
            }    
        }
