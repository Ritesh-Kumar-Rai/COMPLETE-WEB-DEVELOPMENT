Flexbox: flex-basis

The flex-basis property is used to set the initial main size of a flex item. Its effect depends on the direction of the flex container:

  • For Horizontal (Row) Layouts: flex-basis sets the width of the flex item.
  • For Vertical (Column) Layouts: flex-basis sets the height of the flex item.

flex Shorthand Property:

The flex shorthand property is a combination of three properties: flex-grow, flex-shrink, and flex-basis. It's used to define how a flex item will grow, shrink, and its base size within a flex container.

Syntax: 
            .item { 

                   
               flex: <flex-grow> <flex-shrink> <flex-basis>;
               

              }
